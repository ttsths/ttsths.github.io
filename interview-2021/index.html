21年初后端社招面试经历

2021-04-13 interview

背景

2016 年 CS 专业本科毕业后五年，一直在上海，一直在小公司做 Java 开发。迫于上海从 20 年底的房价飞涨，思考对比之后决定回发展，看看外面的机会。毕业这么多年还没有在大厂待过，所以进大厂一直是自己的一个目标。结合自身的情况，给自己制定了一份面试计划，大概分为以下阶段：

  • 知识体系总结回顾
  • 定位目标：我在工程和架构方面的优势是什么？我在软技能方面的长处是什么？我希望找到什么样的工作并加入什么样的团队？
  • 内推面试：感谢 v2 及帮我内推的大佬
  • 面试复盘

准备工作

知知识体系的总结其实是一个长期的过程，但是由于我比较懒，以前的学习都比较零散。我没有很系统地记录和总结，所以在复习的时候花了不少时间。在建立了知识体系之后，我将复习分为以下几个部分：

  • Java 基础&JUC&JVM
  • 过往项目经历
  • 中间件及系统设计
  • 操作系统&计算机网络
  • 算法

准备简历

简历我用的wondercv在线简历网站写的，用的免费的 Markdown 模板（非广告）。重要的是对自己简历上的东西要烂熟于心，熟练的技术栈，项目设计，为什么这么设计，项目的亮点、难点。

总结

面了近 20 家公司，有好多不尽人意的地方，最后能够拿到 offer 还是很幸运的。补充一些小结：

  • 简历里的项目一定要好好准备，熟悉其中的技术栈、项目设计以及项目的亮点和难点。

  • 如果想进入大厂，算法是必须要好好刷一刷的。我推荐以下资源：

    • 《编程珠玑》
    • 《剑指 offer》
    • CodeTop
    • 平时一定要多思考和总结，不积跬步无以至千里
